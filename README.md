  ## Description
  The objective was to build a software engineering team generator command line application. The application prompts the user for information about the team members. The user can create numerous team members with job titles such as manager, engineer, and intern. This generator will allow a manager to generate a webpage that displays the teams basic info and allows quick assess to emals and Github profiles.

  ## Installation
  User must run npm install in order to install pacakages such as inquirer. This prompts the users for email, id, and etc. Once install check package.json for dependecies such as jest and inquirer.
